## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install:
* Python 3.8
Dependencies:
* Discord
```sh
pip3 install discord
```
* asyncio
```sh
pip3 install asyncio
```

### Installation

1. Clone the repo
```sh
git clone git@github.com:Nafiros/B-HUB-500-Nafee.git
```
2. Launch the program with python 3.8
```sh
python3 main.py
```
